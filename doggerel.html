doggerel

doggerel is a chapbook of, well, doggerel (non-serious formal poetry). You can read it one of three ways:

  • You can read it online here, formatted as a webpage.
  • You can find a portable PDF copy, formatted for screen reading, here.
  • You can find a version that is formatted for printing on letter-size paper here, if you would like to create a physical copy. Make sure to print on both sides and select “flip on short end.”
